# Contributing to blatt

Thanks for your interest in contributing!

## How to contribute

1. Fork the repo
2. Create a feature branch (`git checkout -b feature/my-change`)
3. Make your changes
4. Run the tests: `bash tests/test_blatt.sh`
5. Commit with a clear message
6. Open a pull request

## Guidelines

- Keep it pure bash — no external dependencies beyond macOS built-ins
- Run the test suite before submitting
- Keep diffs focused — one feature or fix per PR

## Reporting Bugs

Open an [issue](https://github.com/MichaelGajda/blatt/issues) with:
- What you ran
- What you expected
- What happened instead
- Your macOS version (`sw_vers`)
